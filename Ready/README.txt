======================================== 
DESCRIPTION 
======================================== 
Music Lyrics Fetcher is a desktop application that 
automatically adds lyrics to your audio files. 
 
It supports both MP3 and FLAC formats and uses 
multiple online sources to find the best matching 
lyrics for your songs. 
 
======================================== 
FEATURES 
======================================== 
• Supports MP3 and FLAC audio formats 
• Two lyrics sources: Lyrics.ovh and Lrclib.net 
• Adjustable source priority order 
• Auto-saves your folder settings 
• Clean and simple interface 
• Real-time progress tracking 
• Color-coded execution log 
• No API tokens required 
 
======================================== 
HOW TO USE 
======================================== 
1. Launch the program 
2. Click "Browse..." to select folder with audio files 
3. Adjust source priority (optional) 
4. Click "START LYRICS SEARCH" 
5. Watch the progress in the log window 
6. Wait for completion 
 
======================================== 
REQUIREMENTS 
======================================== 
• Windows 
• No additional software needed 
• No API tokens or registration required 
• Installed Python
